# StudyFlow 🚀

A web app to track your learning progress from YouTube playlists and courses.

## ✨ Features
- Add courses (YouTube playlists)
- Track progress with checklists
- Organize learning by days
- Clean and simple UI

## 🛠 Tech Stack
- Next.js
- TypeScript
- Tailwind CSS

## 🚀 Getting Started

```bash
npm install
npm run dev
```
